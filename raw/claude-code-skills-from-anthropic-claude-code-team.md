## How Anthropic Builds Claude Code Skills

\
Skills are one of the most used extension points in Claude Code — and Anthropic runs hundreds of them internally. But most people think skills are just markdown files. They are not. A skill is a folder: scripts, templates, gotchas, data, and references that Claude can discover and use at runtime.\
This guide breaks down the nine skill types Anthropic uses, the three-level system that keeps context lean, and the design principles that make the difference between a skill that works and one that transforms a workflow. It includes interactive quizzes and a hands-on checklist to get you started building your own.
