## Key Thesis

[[Claude Desktop]] in 2026 is a multi-mode platform with distinct capabilities for different work patterns. Understanding when to use Chat, Artifacts, Cowork, Projects, and Plugins prevents the common mistake of treating Claude as a generic chatbot.

## Important Claims

The guide identifies six core feature areas of [[Claude Desktop]] as of April 2026:

1. **Chat** serves as the general-purpose entry point. Key upgrades include persistent **Memory** across conversations (no more repeating preferences), **Extended Thinking** that shows reasoning chains, and web-grounded Search for real-time information.

2. **Artifacts** render interactive outputs -- websites, budget trackers, charts -- in a side panel. Now available on mobile (iOS/Android) with inline visualizations appearing directly in conversations.

3. **Cowork** is the "digital employee" mode. As of March 2026, [[Claude]] can point, click, and navigate the user's desktop (macOS; Windows coming). The **Dispatch** feature lets users start tasks on desktop and monitor from mobile.

4. **Projects** function as specialized folders with persistent internal knowledge (style guides, API docs, instructions). Team and Enterprise plans support organization-wide sharing.

5. **Excel and PowerPoint Add-ins** enable cross-app context: build a financial model in Excel, then ask Claude to create a summary slide in PowerPoint without re-explaining the data.

6. **Plugins and Skills** extend functionality. Plugins are one-click toolkits (e.g., Marketing, Legal packs). Skills are procedural "expert packs" that teach Claude how to perform specific workflows.

The comparison table distinguishes memory scope across modes: Chat uses global memory, Projects use siloed memory, Cowork uses session-based CLAUDE.md instructions with auto-memory persistence.
