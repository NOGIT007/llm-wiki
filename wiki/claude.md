## Overview

Claude is an AI assistant built by [[Anthropic]]. It is the core model powering both [[Claude Desktop]] and [[Claude Code]], and is referenced across roughly 19 sources in this corpus covering prompting, agents, and reliability.

## Key Features

- **Extended Thinking** -- Allows Claude to reason step-by-step before responding, improving accuracy on complex tasks.
- **Memory** -- Persistent context injection so Claude retains user preferences and constraints across conversations.
- **Projects** -- Scoped workspaces that bundle instructions, files, and conversation history.
- **Cowork Mode** -- Collaborative workflow where Claude asks clarifying questions and iterates alongside the user.
- **Artifacts** -- Structured outputs (code, documents, diagrams) rendered inline during conversation.

## Role in the Corpus

Claude appears as both the subject of [[Prompt Engineering]] techniques (memory injection, reverse prompting, constraint cascades, role stacking, [[Verification]] loops) and as the runtime for agentic workflows (skills, agent swarms, CLAUDE.md project files). Sources frequently address managing [[Hallucinations]] and building [[Reliability]] into Claude-based systems.
